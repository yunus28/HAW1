Projekt: Bildbearbeitung

In diesem Projekt koennen Algorithmen zur Bildbearbeitung implementiert
werden. Einige sind vorgegeben, andere sollen hinzugefuegt werden.
Es koennen Bilder geladen, dargestellt und manipuliert werden.

Die Klasse SWBild definiert ein einfaches Bildformat fuer Schwarz-Weiss-Bilder
sowie die Operationen auf diesem Bildformat.
    
Startpunkt: Erzeuge ein Exemplar der Klasse SWBild über den Konstruktor,
der einen Parameter bekommt. Dieser Parameter ist ein Dateiname fuer
eine Bilddatei. Im Projektverzeichnis liegt ein Unterverzeichnis
'bilder' mit einigen Bildern, die bearbeitet werden koennen. Als Parameter
kann also beispielsweise "bilder/kiss.gif" angegeben werden und
das entsprechende Bild wird geladen.

Alternativ koennt ihr auch den parameterlosen Konstruktor benutzen, um mit 
einem interaktiven Dateidialog eine Bilddatei auszuwaehlen.
    
In diesem Projekt soll der Umgang mit Schleifen und Arrays geuebt werden.
